این پروژه یک ماشین حساب ساده با رابط گرافیکی (Swing) است که امکانات پایه را دارد:

عملیات چهارگانه: جمع، تفریق، ضرب، تقسیم

اعداد اعشاری، علامت منفی (+/-)

دکمهٔ پاک‌سازی (C) و حذف آخرین کاراکتر (<-)

تاریخچهٔ محاسبات (History)

نحوه اجرا

ترمینال (PowerShell) را باز کن.

به پوشهٔ src برو:

cd "C:\Users\Lenovo\Desktop\AP_Calculator\src"


کامپایل:

javac -d ..\out main.java


اجرا:

cd ..\out
java main
